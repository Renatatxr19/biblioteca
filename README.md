# Biblioteca Digital - Landing Page

Este repositório contém uma **Biblioteca Digital** desenvolvida como uma landing page interativa, organizada em seções para fácil navegação e visualização dos livros disponíveis.

---

## Estrutura da Landing Page

A landing page é organizada em seções:

1. **Início**
   - Seção inicial com destaque visual (hero), apresentando a biblioteca digital e uma breve introdução.
   
2. **Recomendados**
   - Mostra os livros de **maior demanda**, com cards contendo imagem, título, categoria e indicação se é gratuito.
   
3. **Categorias de Livros**
   - Menu lateral com links para diferentes categorias:
     - Livros Infantis
     - Livros Didáticos
     - Livros de Cursos
     - Livros Escolares
   - Cada categoria exibe os livros correspondentes na área principal.
   
4. **Sobre a Biblioteca**
   - Breve descrição sobre o objetivo da biblioteca digital, incluindo missão e foco em livros gratuitos e atualizados.
   
5. **Contatos**
   - Informações de contato e links para redes sociais, caso aplicável.
   
6. **Rodapé**
   - Informações finais e créditos.

---

## Tecnologias Utilizadas

- **HTML5** – Estrutura da página
- **CSS3** – Estilização da landing page e layout responsivo
- **Fontes Google Fonts** – Tipografia personalizada
- **Material Symbols** – Ícones para navegação e interação
- **Imagens** – Capas de livros em cards

---

## Funcionalidades

- Menu lateral fixo para navegação entre categorias.
- Cards interativos para cada livro, com hover animation.
- Seções de destaque para livros de maior demanda.
- Layout responsivo que se adapta a diferentes tamanhos de tela.
- Rodapé com informações adicionais.

---

## Estrutura do Código

- **index.html** – Arquivo principal da landing page
- **style.css** – Arquivo de estilos CSS
- **assets/** – Imagens de livros e outros recursos visuais
